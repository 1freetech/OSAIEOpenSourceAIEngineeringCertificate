# OSAIE Certificate Page

This project contains a custom HTML certificate page for the OSAIE credential.

## Credential Name

Open Source AI Engineer Certification

## Overview

The certificate was designed for BitcoinVersus.Tech using a neon black visual theme with a custom retro handheld badge inspired by vintage handheld gaming devices. The page is intended for portfolio use, website display, and print to PDF export.

## Main Features

- Custom HTML and CSS certificate layout
- Neon green and black BitcoinVersus.Tech styling
- Centered title, badge, and metadata layout
- Retro pixel-style handheld badge built directly in SVG
- Consolas based technical typography for titles and metadata
- Ubuntu Mono styling for recipient name
- Print friendly format with Save PDF support
- Credential ID field including Bitcoin block reference
- Program Summary and Proof of Verification modules

## Usage

Open the HTML file in a browser.

To export:
1. Open the file in browser
2. Click `Print / Save PDF`
3. Save as PDF
